Démarrage

Premier cas

Il n'existe aucun fichier de sauvegarde.
L'utilisateur est invité à rentrer un nom pour sa bibliothèque de films.
Ce nom ne doit ni être vide, ni contenir de points.
Après cela, l'utilisateur peut clicker sur le bouton 'OK'.
Retour vers le haut

Deuxième cas

Il existe déjà un fichier de sauvegarde.
Le programme lance la bibliothèque sauvegardée, l'utilisateur n'a rien à faire.
Retour vers le haut

Troisième cas

Il existe plus d'un fichier de sauvegarde.
L'utilisateur est invité à choisir une bibliothèque parmi la liste proposée.
Après cela, l'utilisateur peut clicker sur le bouton 'OK'.
Retour vers le haut

Afficher et modifier les films

Afficher un film

Pour afficher un film, il suffit à l'utilisateur de cliquer sur la jaquette du film choisi.
Une fois cela fait, les informations connues sont affichées.
Il suffit de fermer la nouvelle fenêtre pour retourner à la bibliothèque.
Retour vers le haut

Modifier un film

L'utilisateur doit afficher les informations du film à modifier.
Ensuite, il doit cliquer sur le bouton 'Modifier' et mettre
les informations à jour.
Une fois les modifications terminées, l'utilisateur peut quitter la fenêtre en cliquant
sur le bouton 'Sauvegarder'. Les modifications seront enregistrées dans le
fichier de sauvegarde.
Retour vers le haut

Ajouter ou supprimer des films

Ajouter un film

Premièrement, l'utilisateur doit activer le bouton 'Modifier' afin d'afficher
les boutons 'Ajouter' et 'Supprimer'.
Maintenant, l'utilisateur peut appuyer sur le bouton 'Ajouter'.
La fenêtre d'initialisation du film s'afficher et l'utilisateur peut entrer les informations du film.
Une fois le film initialisé, l'utilisateur peut quitter la fenêtre en cliquant
sur le bouton 'Sauvegarder'. Le film sera enregistré dans le fichier de sauvegarde.
Il est à noter que les films se trient automatiquement par date.
Retour vers le haut

Supprimer des films

Après avoir activé le bouton 'Modifier', l'utilisateur peut appuyer sur le bouton 'Supprimer'.
Une nouvelle fenêtre s'ouvre, contenant une liste des films de la bibliothèque.
Une fois le ou les film(s) à supprimer choisi(s), l'utilisateur est invité
de cliquer sur le bouton 'OK'. La fenêtre se fermera et les films seront supprimés
ainsi que toutes leurs données.
Retour vers le haut

Supprimer un film

L'utilisateur peut, soit passer par la méthode longue (voir point précédent),
soit par la version courte.
Il suffit d'effectuer un clic droit sur le film à supprimer.
Le film ainsi que toutes ses données seront supprimées.
Retour vers le haut

Créer une bibliothèque

Pour créer une nouvelle bibliothèque, l'utilisateur peut cliquer sur le bouton
'Créer bibliothèque' qui fermera la fenêtre courante et ouvrira une boîte de dialogue
l'invitant à entrer un nom non vide et ne contenant pas de point.
Une fois le nom choisi et le bouton 'OK' déclenché, la bibliothèque est créée et affichée.
Retour vers le haut

Réinitialiser

Il suffit à l'utilisateur d'appuyer sur le bouton 'Réinitialiser' et la bibliothèque
courante sera immédiatement réinitialisée.
Retour vers le haut

Supprimer une bibliothèque

L'utilisateur n'a qu'à appuyer sur le bouton "Supprimer bibliothèque", et la bibliothèque
courante sera supprimée.
La fenêtre se fermera et le programme se relancera.
Retour vers le haut

Recherche

Il suffit à l'utilisateur de commencer à écrire dans la barre de recherche,
et les films contenant le texte entré s'afficheront automatiquement.
Pour retrouver la vue normale, il suffit d'effacer le texte entré.
Retour vers le haut

Raccourcis

Il existe plusieurs raccourcis dans la fenêtre principale.
Pour activer les boutons 'Modifier', 'Réinitialiser', 'Ajouter' et 'Supprimer',
l'utilisateur peut exécuter ces raccourcis: alt + m, alt + r, alt + a et alt + s.
Il reste un autre raccourcis: alt + h pour ouvrir l'aide.
